# CollegeFolder

A web app to manage your college semesters, courses, todos, and exams.

## Tech Stack
- Backend: FastAPI (Python)
- Frontend: HTML + CSS + Vanilla JS
- Database: PostgreSQL

## Features
- Login / Register
- Manage semesters and courses
- Todo list for each course
- Exam dates with D-3 warning

## Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Set up `.env` with your database URL
4. Run: `uvicorn main:app --reload`
